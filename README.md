SimBanker
=========

Part of Brighton Hackathon 2013.

Learn how to be a banker in the 21st century! Make your billions! But keep an eye on that market...

## Instructions

1. Clone repo
2. Install node (if haven't already)
3. Install [bower](http://bower.io/)
4. Run `npm install`
5. Run `bower install`
6. Run `grunt server`
